## 安装
为了方便自己开发新项目,收集项目中常用功能进行封装并用Composer进行下载

`composer require phpcpj/phplib dev-master` 

## 文件目录介绍

### src - 类库存放文件

**Curl.php**

用`curl`进行双向认证的类

**File.php**
`downloadFile`  下载本地或网络资源

`createZip`     压缩文件



### test
测试类库文件,分别对应src中的类库

### doc 
php 基础知识

### html
前端知识存放处
